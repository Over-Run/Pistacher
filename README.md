# Pistachio Interpreter

Interpret the Pistachio language.  
Like [Python](https://www.python.org), but it can interaction with Java!

## Milestone

- 0.1.0
    - ```
      println "content"
      ```
- 0.2.0
    - ```
      name:
          println "content"
      call name
      ```
- 0.3.0 (WIP)
    - ```
      void name():
          println "content"
      //if __name__ == "main":
      name()
      ```
